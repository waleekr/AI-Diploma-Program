# Unit 3: Cloud and Containerization | السحابة والحاويات
## AIAT 125 - AI Model Deployment

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Deploy models on cloud platforms
- Use Docker for containerization
- Understand Kubernetes basics
- Implement serverless deployment
- Optimize cloud costs

---

## Topics Covered | المواضيع المغطاة

1. **Cloud Platforms**
   - AWS (SageMaker, EC2, Lambda)
   - Azure (ML Studio, Functions)
   - Google Cloud (AI Platform, Cloud Functions)
   - Platform comparison

2. **Docker Containerization**
   - Docker basics
   - Creating Dockerfiles
   - Building images
   - Running containers

3. **Kubernetes Basics**
   - Kubernetes concepts
   - Deployments and services
   - Scaling applications
   - Load balancing

4. **Serverless Deployment**
   - Serverless functions
   - AWS Lambda
   - Azure Functions
   - Google Cloud Functions

---

**Unit Duration:** 2 weeks  
**Difficulty:** Intermediate  
**Prerequisites:** Unit 1-2 completion
