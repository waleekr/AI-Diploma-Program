# Unit 2: Generative Adversarial Networks (GANs) | الشبكات التوليدية التنافسية
## AIAT 124 - Generative AI

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand GAN architecture
- Build generator and discriminator networks
- Train GANs effectively
- Understand GAN variants (DCGAN, StyleGAN)
- Apply GANs to image generation

---

## Topics Covered | المواضيع المغطاة

1. **GAN Fundamentals**
   - Generator network
   - Discriminator network
   - Adversarial training
   - Loss functions

2. **GAN Training**
   - Training process
   - Common challenges
   - Mode collapse
   - Training stability

3. **GAN Variants**
   - DCGAN (Deep Convolutional GAN)
   - Conditional GANs
   - StyleGAN
   - Progressive GANs

4. **Applications**
   - Image generation
   - Image-to-image translation
   - Super-resolution
   - Data augmentation

---

**Unit Duration:** 3 weeks  
**Difficulty:** Advanced  
**Prerequisites:** Unit 1 completion, strong deep learning background
