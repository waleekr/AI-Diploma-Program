## ğŸ“ **Notes**

1. **File Naming Convention:**
   - Descriptive names (e.g., `fairness_metrics_comparison.png`)
   - Lowercase with underscores
   - Clear indication of content

2. **File Format:**
   - All visualizations are PNG files
   - High resolution (300 DPI)
   - Suitable for presentations and reports

3. **Storage Location:**
   - Always in the `examples/` directory of each unit
   - Same directory as the generating script/notebook

4. **Path Issues:**
   - Some scripts use relative paths (may cause issues)
   - Fixed scripts use `os.path.dirname(__file__)` for Python
   - Notebooks use `get_examples_dir()` helper function
