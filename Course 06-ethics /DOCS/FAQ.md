# Frequently Asked Questions | الأسئلة الشائعة

## General Questions | الأسئلة العامة

### Q: What is this course about?
**A:** This course explores the ethical implications and responsibilities associated with Artificial Intelligence (AI). You'll learn about bias, justice, privacy, transparency, accountability, and governance in AI systems.

### Q: Do I need prior AI/ML knowledge?
**A:** Basic understanding is helpful but not required. The course focuses on ethical considerations rather than technical AI implementation.

### Q: What programming knowledge do I need?
**A:** Basic Python programming knowledge (variables, functions, classes) is recommended. You should be comfortable reading and modifying Python code.

## Installation Questions | أسئلة التثبيت

### Q: Which Python version should I use?
**A:** Python 3.8 or higher is required. Python 3.10 or 3.11 is recommended.

### Q: Do I need to install all libraries?
**A:** Yes, all libraries in `requirements.txt` are needed for the examples and exercises.

### Q: What if I get library conflicts?
**A:** Use a virtual environment. See `INSTALLATION_GUIDE.md` for detailed instructions.

### Q: Can I use Anaconda instead of pip?
**A:** Yes, you can use conda:
```bash
conda install --file requirements.txt
```

## Technical Questions | الأسئلة التقنية

### Q: Why are there both .py and .ipynb files?
**A:** Both formats are provided for flexibility. Use Jupyter notebooks (.ipynb) for interactive learning, or Python scripts (.py) for direct execution.

### Q: How do I run the examples?
**A:** For notebooks: `jupyter notebook unit1-ethics-foundations/examples/01_ethical_frameworks.ipynb`  
For Python scripts: `python unit1-ethics-foundations/examples/01_ethical_frameworks.py`

### Q: The visualizations aren't showing up
**A:** Make sure matplotlib is installed and you're running the code in an environment that supports display (Jupyter notebook or with `plt.show()`).
