## Part 1: Privacy Principles (15 points)

### Question 1 (5 points)
What is data minimization?
- A) Using all available data
- B) Collecting only necessary data
- C) Maximizing data collection
- D) Storing all data

**Answer:** B

---

### Question 2 (5 points)
What is purpose limitation?
- A) Using data for any purpose
- B) Using data only for stated purposes
- C) No limitations
- D) Unlimited use

**Answer:** B

---

### Question 3 (5 points)
What does GDPR stand for?
- A) General Data Protection Regulation
- B) General Data Processing Rules
- C) Global Data Protection Rules
- D) General Data Privacy Rules

**Answer:** A

---

## Part 2: Privacy-Preserving Techniques (15 points)

### Question 4 (5 points)
What is differential privacy?
- A) Complete privacy
- B) Mathematical framework for privacy-preserving data analysis
- C) No privacy
- D) Public data

**Answer:** B

---

### Question 5 (5 points)
What is federated learning?
- A) Centralized training
- B) Training models without centralizing data
- C) Public training
- D) No training

**Answer:** B

---

### Question 6 (5 points)
What is k-anonymity?
- A) No anonymity
- B) Privacy technique where individuals cannot be distinguished from k-1 others
- C) Complete anonymity
- D) Public data

**Answer:** B
