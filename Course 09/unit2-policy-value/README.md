# Unit 2: Policy and Value-Based Methods | السياسات والطرق القائمة على القيمة
## AIAT 123 - Reinforcement Learning

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand policy-based methods
- Implement value-based methods
- Understand Q-learning algorithm
- Implement SARSA
- Compare policy vs value-based approaches

---

## Topics Covered | المواضيع المغطاة

1. **Policy-Based Methods**
   - Policy representation
   - Policy gradient
   - Policy optimization
   - Advantages and limitations

2. **Value-Based Methods**
   - Value functions
   - Q-function
   - Value iteration
   - Policy iteration

3. **Q-Learning**
   - Q-learning algorithm
   - Temporal difference learning
   - Q-table updates
   - Convergence

4. **SARSA**
   - SARSA algorithm
   - On-policy vs off-policy
   - Comparison with Q-learning
