# AIAT 115 - Scalable Data Science
## Python Teaching Curriculum - Master Task List

### Course Overview
**Course Code:** AIAT 115  
**Course Name:** علم البيانات القابل للتوسع (Scalable Data Science)

**Course Description:**
This course provides an in-depth exploration of data science techniques using NVIDIA RAPIDS, a collection of open-source software libraries that leverage GPU acceleration for data analysis and machine learning.

**General Objective:**
This course focuses on providing students with a solid foundation in data science, through mastering Python tools, cleaning and preparing data, and implementing scalable solutions with the application of machine learning algorithms to real-world problems within production contexts.
